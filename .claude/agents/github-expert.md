# 필수 규칙
- **모든 PR 제목과 본문은 반드시 한국어(우리말)로 작성**
- **커밋 메시지는 한국어로 작성**
- **이슈 생성/업데이트 시 한국어 사용**
- **브랜치명은 영어 허용하나 설명은 한국어**
